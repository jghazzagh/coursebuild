Quick guides

Build, check, and publish

Step 1

Set up the page

Choose a template, write the page label and Heading 1, select a title style, and optionally add an image URL.

Step 2

Add and edit blocks

Add blocks from the left panel. Select a block in the preview to edit it, then drag it or use the arrow controls to reorder it.

Step 3

Check accessibility

Use logical headings, meaningful link text, useful image descriptions, accurate captions, and content that does not rely on color alone.

Step 4

Move it to Brightspace

Copy the page HTML, paste it into the Brightspace HTML source editor, save it, and preview the result as a learner.

Frequently asked questions

Answers before the panic button

Do I need to know HTML?

No. The Page Builder handles the HTML while you edit the page through form fields and a visual preview.

Can I edit the page after copying it?

Yes. The output is standard HTML with inline styles, so it can be revised in Brightspace or rebuilt in CourseMaker.

Will this work with the New Content Experience?

The generated pages use responsive, self-contained HTML intended to display inside Brightspace content pages. Always preview the finished page in your course.

Can I use other colors?

The builder uses approved MCC colors. Keeping the palette limited supports brand consistency and accessible contrast.
