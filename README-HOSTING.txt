MCC CourseMaker static site
===========================

Upload all five HTML files to the same folder on your web host:

  index.html
  templates.html
  mcc-brightspace-template-builder.html
  widget-builder.html
  support.html

Keep the filenames unchanged so the navigation links continue to work.
Set index.html as the folder's default page. No CSS, JavaScript, fonts, or
image assets are required. Each page contains its own styling and scripts.

The Page Builder and Widget Builder can copy generated HTML to the clipboard
or download it as an HTML file. Their generated Brightspace code also uses
inline styling and does not depend on external CSS or JavaScript files.
